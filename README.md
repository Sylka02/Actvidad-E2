## Compara a 2 numeros
## Descripcion del problema
Este programa en c++ permite comparar dos numeros para saber cual es el menor en caso contrario que indique si son iguales
## Funcionalidad
int x,y;
## Salida
si (x==y) son iguales
si (x<y) x es menor
